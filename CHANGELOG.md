# Changelog

Журнал изменений данных `data/*.jsonl` — источника правды экосистемы avar.me.

Фиксируем здесь **смысловые правки данных** (словарные статьи, формы, метки, корпусы),
а не служебные изменения сайта/шаблонов/CI. Потребители (`forms`, `avar.me`, `corrector`,
`index`) ориентируются по этому файлу, чтобы понимать, что и когда поменялось.

Формат: новые записи сверху, по датам. У каждой записи — дата, краткое описание, затронутый
источник (`av-ru`, `ru-av`, …) и объём правок. Коммит указывается, когда правка уже в `main`.

Крупные правки сопровождаются подробным построчным diff в папке [`changelog/`](changelog/)
(имя файла `ГГГГ-ММ-ДД-<источник>-<тема>.md`) — чтобы корректоры могли сверить каждую статью.

Исходные текстовые словари (`av-ru.dict.txt`, `ru-av.dict.txt`) в репозитории не храним; берём
их при необходимости из [avar-me/avar-russian-dictionary](https://github.com/avar-me/avar-russian-dictionary).

## 2026-06-30 — mukhidinov-library: контекстная нормализация палочки

Пересобран `data/mukhidinov-library.jsonl` после уточнения правил палочки
(CLAUDE.md, раздел «Аварская графика»):

- Палочка теперь стоит **только в диграфах** (после т/г/к/х/ц/ч).
- Регистр палочки совпадает с регистром основы (`ГӀумру`, не `Гӏумру`).
- Цифра `1` после диграф-основы → палочка (`т1ок1ав` → `тӏокӏав`).
- Палочка-глиф вне диграфа → цифра `1` (`Ӏ993` → `1993`, `Ӏ классалдаса` → `1 классалдаса`).
- Удвоение `чII`, `т11`, `тӏӏ` → одиночная палочка (`чӏ`, `тӏ`).

Суммарно: 123 724 палочки против 82 431 в первой версии (в основном за счёт книги
ГъалбацӀова, где типист печатал цифру `1` вместо палочки). Исправлены сотни лет/дат
(`1993`, `2014`, `11.08.43`, `24.01.94`), номеров классов и пр.

Также: `src/build_site.py::normalize_palochka` теперь следует тем же правилам — словари
(`av-ru`, `phrase-av-ru`) ребилдятся одинаково.
